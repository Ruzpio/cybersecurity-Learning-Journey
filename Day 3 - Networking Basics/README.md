## Day 3 — Networking Basics

**Topic:** Network Fundamentals  
**Date:** 18/3/2026

### What I Learned
- **IP Address** – Unique identifier for a device on a network
- **MAC Address** – Hardware-level permanent identifier
- **Ports** – Numbered doors for services (80=HTTP, 443=HTTPS, 22=SSH)
- **DNS** – Resolves domain names to IP addresses

### Lab — Commands Run
```bash
ipconfig /all       # Found IP and MAC address
nslookup google.com # Saw DNS resolution live
```

### Results
- My IPv4: [10.0.2.15]
- My MAC: [08-00-27-DF-9D-0A]
- Google's IP via DNS: [172.217.22.78]

### Key Takeaway
Every attack starts with network reconnaissance. 
Understanding these basics is non-negotiable.
